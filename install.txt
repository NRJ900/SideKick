# Development Requirements for Desktop Sidekick

To develop the Desktop Sidekick application (Tauri v2 + React + Rust) on Windows, you need to install the following tools and dependencies.

## 1. System Requirements
- **OS**: Windows 10 or Windows 11
- **WebView2**: Built-in on Windows 10/11. If missing, download the "Evergreen Bootstrapper" from Microsoft.

## 2. Core Build Tools (Install in this order)

### A. Microsoft Visual Studio C++ Build Tools
1. Download **Visual Studio Build Tools** (2022 recommended) from [visualstudio.microsoft.com](https://visualstudio.microsoft.com/visual-cpp-build-tools/).
2. During installation, select "Desktop development with C++".
3. Ensure the "Windows 10/11 SDK" component is checked on the right side.

### B. Rust Programming Language
1. Download `rustup-init.exe` from [rust-lang.org](https://www.rust-lang.org/tools/install).
2. Run the installer and proceed with the default installation (Option 1).
3. Restart your terminal (PowerShell/CMD).
4. Verify installation: `rustc --version` and `cargo --version`.

### C. Node.js & npm
1. Download the **LTS version** (v18 or v20+) from [nodejs.org](https://nodejs.org/).
2. Install with default settings.
3. Verify installation: `node --version` and `npm --version`.

## 3. Project Dependencies (Environment Setup)

Once the tools above are installed, you will need to set up the project.

### Tauri CLI
It is recommended to use the Tauri CLI via your Node package manager, but you can also install it globally:
```powershell
cargo install tauri-cli --version "^2.0.0-rc" --locked
```
*Note: Tauri v2 is currently in beta/RC. The project requires v2.*

## 4. Specific Libraries (Will be installed via project config)
These do not need to be downloaded manually, but will be fetched when you build the project.

**Frontend (npm/pnpm/yarn):**
- React ^18
- TypeScript
- TailwindCSS ^3
- @tauri-apps/api ^2

**Backend (Rust Crates):**
- tauri ^2.0
- tauri-plugin-global-shortcut
- tauri-plugin-opener
- tauri-plugin-fs
- serde, serde_json
- tokio
- reqwest
- aes-gcm (Encryption)
- clipboard-win (Clipboard management)
- windows (Win32 API)

## 5. Development Vs Code Extensions (Recommended)
- **rust-analyzer**: Essential for Rust development.
- **Tauri**: Official Tauri extension.
- **ESLint / Prettier**: For TypeScript/React code quality.
